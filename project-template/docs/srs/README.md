# SRS / PRD

The detailed specification for the loaded engagement lives here.

The methodology shape must be chosen and recorded in `docs/decisions/`
before drafting. Valid shapes:

- SRS-only: rename this folder to `srs/` and draft from
  `docs/templates/srs.md`.
- PRD + stories: rename this folder to `prd/` and draft from
  `docs/templates/prd.md`.
- Stories-only: leave this folder empty and work directly under
  `../stories/`. Record the decision to skip SRS/PRD.

Sign-off evidence is recorded in `../validation/`, not inside the spec body.
